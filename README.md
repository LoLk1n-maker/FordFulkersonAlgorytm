# FordFulkersonAlgorytm
Реализация алгоритма Форда фалкерсона для рещения задачи трассировки кабельных соединений


Введение:
Задача трассировки кабельных соединений в каналах представляет собой задачу поиска максимального потока в сети и заключается в определении оптимальных маршрутов прокладки кабелей между заданными узлами с учетом ограниченной пропускной способности каналов. К решению данной задачи сводится проектирование и оптимизация различных инженерных систем, включая проектирование электросетей и систем энергоснабжения, телекоммуникационных систем, структурированных кабельных систем, компьютерных сетей, а также планирование инфраструктуры центров обработки данных. Данная задача является одной из базовых в системах автоматизированного проектирования (САПР) и требует применения эффективных алгоритмов теории графов [3, 6].
